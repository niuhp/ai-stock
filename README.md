# AI Stock - A股智能分析系统

## 项目简介

AI Stock 是一个基于人工智能的A股股票分析系统，通过机器学习和大数据分析技术，为投资者提供智能化的股票分析、买卖点推荐和投资决策支持。

## 技术栈

- **后端框架**: Spring Boot 3.x
- **开发语言**: Java 17
- **数据库**: MySQL 8.0 + Redis
- **消息队列**: RabbitMQ / Kafka
- **AI/ML**: Python集成 (TensorFlow/PyTorch) 或 Java ML库
- **数据采集**: 第三方股票API（如：东方财富、同花顺、Tushare等）
- **缓存**: Redis
- **搜索引擎**: Elasticsearch（可选）
- **文档**: Swagger/OpenAPI
